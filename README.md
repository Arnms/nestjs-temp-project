## Description

- Base = NestJS + MySQL + TypeORM
- Auth = JWT + Passport

## Getting Started

### Prerequisites

- node >= 18.15.0
- npm >= 9.5.0
- Nest CLI >= 10.1.9

### Dependencies

-

### Structure

```
- src
  - auth
  - common
  - database
  - modules
  - services
  - app.controller.ts
  - app.module.ts
  - main.ts
```

### Installing

```
yarn
```

### Run Command

- build

```
yarn build
```

- run dev mode

```
yarn start:dev
```

- run debug mode

```
yarn start:debug
```

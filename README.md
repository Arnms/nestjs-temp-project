## Description

- Base = NestJS + MySQL + TypeORM
- Auth = JWT + Passport

## Getting Started

### Prerequisites

- Yarn >= 1.22.19
- Nest CLI >= 9.1.5

### Dependencies

-

### Structure

```
- src
  - auth
  - common
  - database
  - modules
  - services
  - app.controller.ts
  - app.module.ts
  - main.ts
```

### Installing

```
yarn
```

### Run Command

- build

```
yarn build
```

- run dev mode

```
yarn start:dev
```

- run debug mode

```
yarn start:debug
```
